Hotel Database System

A simple hotel management application built with C# Windows Forms using a database-first approach with Microsoft SQL Server.

Overview

Desktop application with a Windows Forms UI

Database-first architecture

Connected to MS SQL Server

Tech Stack

C# (.NET)

Windows Forms

Microsoft SQL Server

ADO.NET / Entity Framework (Database First)

Purpose

The project demonstrates a basic hotel database system with a graphical user interface, focusing on structured data access and relational database design.

Status

Educational / prototype project.
